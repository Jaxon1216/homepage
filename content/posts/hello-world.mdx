## 为什么要迁移？

之前用 Hexo 搭建博客，虽然简单好用，但随着需求增长（在线简历、路线图、动态交互），静态模板的局限性越来越明显。

## 新技术栈

- **Next.js 14** (App Router) - React 全栈框架
- **TypeScript** - 类型安全
- **Tailwind CSS** - 原子化样式
- **Framer Motion** - 流畅动画
- **MDX** - Markdown + React 组件

## 代码示例

```typescript
const greeting = (name: string): string => {
  return `Hello, ${name}!`;
};

console.log(greeting("World"));
```
